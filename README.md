# ChatSphere — Real-Time Messaging System

## 📌 Overview
ChatSphere is a real-time messaging system built from scratch using Java socket programming. The project demonstrates how multiple clients communicate through a centralized server using TCP connections while maintaining synchronized conversations.

## 🎯 Project Objectives
* Understand client-server communication
* Implement real-time messaging
* Handle multiple users simultaneously
* Persist chat messages using a database
* Explore distributed system concepts
* Integrate intelligent message analysis

## 🏗️ System Architecture
Clients <-> Chat Server <-> Database
